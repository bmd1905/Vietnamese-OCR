# PaddleOCR VietOCR for Vietnamese Texts
Ongoing Project

This is a project about Optical Character Recognition (OCR) in Vietnamese texts by using PaddleOCR and VietOCR.

# Outline
1. Text Preprocessing
2. Text Detection
3. Text Alignment
4. Text Recognition
5. OCR Error Correction
